**Course:** Data Analytics II  
**Professor:** Prof Bongo Adi

## About This Project
This study investigates workplace experience and employee 
retention intentions among 100 professionals in Nigerian 
legal and professional services organisations.

## Five Techniques Applied
1. Exploratory Data Analysis (EDA)
2. Data Visualisation
3. Hypothesis Testing
4. Correlation Analysis
5. Logistic Regression
